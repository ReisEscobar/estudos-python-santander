## 📚 O que aprendi sobre Loops
Os loops servem para repetir tarefas. Existem dois tipos principais:

* **For**: Usado quando sabemos o limite (ex: percorrer uma lista).
* **While**: Usado enquanto uma condição for verdadeira.

### Exemplos de Comandos de Controle:
1. **Break**: Para o loop na hora.
2. **Continue**: Pula para a próxima volta.
3. **Pass**: Não faz nada, serve de reserva.

> "Programar é a arte de dizer ao computador o que fazer, repetidamente."
